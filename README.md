# Student Attendance Management System

## Project Description
The **Student Attendance Management System** is a Java-based application developed using **JDBC and MySQL**. It is designed to simplify and automate the process of managing student attendance in an efficient and organized way. The system allows users to add student details, mark attendance (Present/Absent), and view attendance records stored in a database.

---

## Features
- Add new student records
- Mark attendance (Present/Absent)
- View student attendance records
- Store data securely in MySQL database
- Simple and user-friendly interface (Console-based / GUI if applicable)

- 
##  Technologies Used
- Java (Core Java)
- JDBC (Java Database Connectivity)
- MySQL Database
- SQL Queries
